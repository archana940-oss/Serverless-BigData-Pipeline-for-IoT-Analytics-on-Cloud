# 🚀 Serverless Big Data Pipeline for IoT Analytics on Cloud

## 📌 Project Overview

This project demonstrates a cloud-native serverless big data pipeline for processing, analyzing, and visualizing IoT sensor data using Amazon Web Services (AWS). The system leverages managed serverless services to reduce infrastructure management while providing scalable, secure, and cost-effective data analytics.

---

## 🎯 Objectives

- Build a serverless IoT analytics pipeline.
- Store IoT sensor data using Amazon S3.
- Process data using AWS Glue.
- Analyze data using Amazon Athena.
- Execute serverless functions using AWS Lambda.
- Visualize analytical results.
- Apply cloud security using AWS IAM.

## ⚙️ Workflow

1. Upload IoT Dataset to Amazon S3.
2. Create AWS Glue Database.
3. Run AWS Glue Crawler.
4. Query data using Amazon Athena.
5. Process events using AWS Lambda.
6. Visualize analytics using dashboards.
7. Secure cloud resources with IAM.
